# Care-Connect
**Hospital Management Website**

This is a web-based Hospital Management System designed to simplify the process of finding doctors and booking appointments online. The project provides a user-friendly interface for patients to search for doctors based on specialization and schedule appointments efficiently.

**Features**

Search doctors by name or specialization

Book and manage doctor appointments

Responsive and interactive user interface

Secure data storage using MongoDB
